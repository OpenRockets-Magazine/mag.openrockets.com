# OpenRockets Magazine

A Wall Street Journal-inspired news magazine website focused on space exploration and rocket technology.

## Overview

This is a modern, responsive frontend clone inspired by the Wall Street Journal's design aesthetics. The website features a clean, professional layout with serif typography, organized content sections, and interactive elements—all built with pure HTML, CSS, and JavaScript without any backend dependencies.

## Features

- **Responsive Design**: Fully responsive layout that adapts seamlessly to desktop, tablet, and mobile devices
- **WSJ-Inspired Typography**: Classic serif fonts for headlines with clean, readable layouts
- **Interactive Navigation**: Mobile-friendly hamburger menu with smooth transitions
- **Article Grid Layout**: Well-organized content sections including featured articles, news grid, and opinion pieces
- **Dynamic Elements**: 
  - Current date display
  - Reading progress indicator
  - Smooth scrolling
  - Hover effects on articles
  - Sticky header with auto-hide on scroll
- **Professional Footer**: Comprehensive footer with links organized by category

## Tech Stack

- **HTML5**: Semantic markup for better accessibility and SEO
- **CSS3**: Modern styling with Flexbox and Grid layouts, responsive media queries
- **JavaScript (ES6+)**: Pure vanilla JavaScript for interactivity (no frameworks)

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local web server (optional but recommended)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/OpenRockets-Magazine/mag.openrockets.com.git
cd mag.openrockets.com
```

2. Open the website:

**Option A: Using Python's built-in server**
```bash
python3 -m http.server 8080
```
Then open `http://localhost:8080` in your browser.

**Option B: Direct file access**
Simply open `index.html` in your web browser.

## Design Philosophy

This project follows the Wall Street Journal's design principles:
- **Typography**: Emphasis on serif fonts for headlines, creating a traditional newspaper feel
- **Layout**: Grid-based design with clear visual hierarchy
- **Color Scheme**: Minimal color palette focusing on black, white, and subtle grays
- **Whitespace**: Generous spacing for improved readability
- **Responsiveness**: Mobile-first approach ensuring accessibility across all devices
